# socketio-live-balls
SocketIO Live Balls

Live demo on heroku
[https://socketio-live-balls.herokuapp.com/](https://socketio-live-balls.herokuapp.com/)

## Local Installation
Clone this repo and run commands below.

> npm install

> npm install -g nodemon

> npm run start:dev

and go to [http://localhost:3000](http://localhost:3000)

enjoy! :heart_eyes:
